DESCRIPTION:
This is my version of a key-value store program. It is a simple, store-themed inventory program.

USAGE:
Open the key_value.py file in your IDE, or find it in your file system and run the command: python key_value.py.
It runs on Python, so make sure you have Python installed on your system before using it.

FEATURES:
1. Allows users to create their own products using the Product class.
2. Gives users the option to store their products using a dictionary.
3. Allows users to save their inventory in a .json file for later use.
4. Lets users search for their products by word, not just by their key.

HOW IT WORKS:
The program uses a simple menu system where users can view inventory, add products, remove products, search for items, and save or load their inventory from a .json file.

REQUIREMENTS:
- Python 3
- A text editor or IDE such as VS Code
